## 💼 Sobre el Proyecto

Este repositorio aloja mi **Portafolio Web y Currículum Digital**, una plataforma optimizada y de alto rendimiento diseñada para presentar de manera interactiva más de 19 años de trayectoria en el sector analítico y de control. El sitio no solo funciona como una carta de presentación corporativa, sino también como un entorno para demostrar mis capacidades técnicas en desarrollo frontend moderno y arquitectura web orientada al usuario.

Su enfoque principal es exhibir proyectos complejos que involucran **Data Science, Machine Learning, Automatización de Procesos (IA/RAG) y Business Intelligence**, estructurados de forma que reclutadores y socios estratégicos puedan filtrar y evaluar soluciones de manera ágil.

## 🚀 Características Clave

*   ⚡ **Rendimiento Óptimo:** Arquitectura estática sin sobrecarga de frameworks, garantizando una carga inmediata y una excelente puntuación en Core Web Vitals.
*   🎛️ **Filtro de Proyectos Dinámico:** Interfaz de usuario intuitiva que permite segmentar proyectos de manera inmediata (*Data Science, AI & Automation, Web Apps*).
*   📱 **Enfoque Mobile-First:** Diseñado críticamente para adaptarse de manera limpia al formato vertical de los smartphones.
*   💼 **Orientación Corporativa:** Tono estrictamente profesional adaptado a las demandas de auditoría, gobernanza de datos y cumplimiento normativo.
